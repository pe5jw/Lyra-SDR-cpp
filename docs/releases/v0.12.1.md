# Lyra v0.12.1

**Hardware / scope:** Hermes Lite 2 / 2+ over HPSDR Protocol 1. RX, plus
CW / SSB / AM / DSB / SAM / FM and digital (TCI / VAC) transmit. **Windows
only** for now (Linux / macOS planned, not yet buildable).

A polish release: new recovery tools, a cleaner set of on-screen controls, and
much quieter logs. Nothing about your operating changes — your settings,
profiles, and calibrations are untouched.

## Maintenance & recovery (Settings → Backup & Restore)

Two "when things go sideways" tools on the Backup & Restore tab, both taking
effect on the next launch:

- **Rebuild FFT optimization** — Lyra tunes its FFT math to your CPU on first
  run and caches it. Swap the processor, or move the drive to another machine,
  and this rebuilds it so it re-tunes to the new hardware.
- **Reset to factory defaults** — the panic button: put every setting back to
  how a fresh install behaves. A snapshot is saved **first**, so you can undo it
  from the Snapshots list. Your saved snapshots and the FFT cache are kept.

## Redesigned on-screen controls

The panel sliders, drop-downs, and spin boxes are now Lyra-drawn at a fixed
size instead of borrowing the Windows look:

- **Sliders** — a clean white track with a compact cyan knob.
- **Drop-downs** (Mode, RX/TX BW, Rate, Zoom, …) — the list now opens as its own
  window, so long lists (like Mode) are never clipped by a panel edge, and the
  **mouse wheel** cycles the selection.
- **Spin boxes** — brighter value text, and now **click to type an exact number**
  in addition to the arrows and the mouse wheel.

These render identically on any machine and can't be resized by the system
theme.

## Quieter logs

The diagnostic log no longer fills with hundreds of harmless "does not support
customization" style advisories per launch, so real messages are actually
readable now — a help both for you and for anyone sending in a log for support.

## Small fixes

- The space-bar-as-PTT and CAT settings each got a minor internal cleanup.
- A stray weather diagnostic that logged at "warning" level is now a debug line.

---

**Upgrade note:** drop-in — install over your existing Lyra, everything carries
over.

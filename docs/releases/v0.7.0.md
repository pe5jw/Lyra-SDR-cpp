# Lyra v0.7.0

**Hardware / scope:** Hermes Lite 2 / 2+ over HPSDR Protocol 1. RX, plus
CW / SSB / AM / DSB / SAM / FM and digital (TCI / VAC) transmit. **Windows
only** for now (Linux / macOS planned, not yet buildable).

The headline of this release is a new **Tuner memory** — a built-in tuning
memory for manual antenna tuners, so you can return to a known-good match
without paper sheets or a third-party app. Plus FM transmit polish, a global
tooltips on/off switch, and a tidier header chip strip.

## Tuner — manual-ATU tuning memory (header → Options → Tuner)

A new **Tuner** panel remembers the Input / Output / Inductor settings of
your **manual** antenna tuner, frequency by frequency, so a re-tune starts
from the right place instead of from scratch. Pure memory — it doesn't drive
or read the tuner, it remembers what *you* dialled in.

- **Three named antennas** (Beam / Dipole / Vertical by default, renameable),
  each with its own frequency-keyed table of Input / Output / Inductor
  settings.
- **Live, colour-coded SWR** in the header — green ≤ 1.5, amber 1.6–2.4,
  red ≥ 2.5 — so you watch the match as you adjust.
- **Nearest-point bracket view.** Instead of a scrollable list of hundreds of
  saved combinations, the panel shows only the points that *bracket* your
  dial: the one just below and the one just above (and the exact row when
  you're sitting on a saved frequency). On an unsaved frequency you get the
  saved setting one notch down and one notch up — an instant starting
  position. A **±x.x kHz** badge tells you how far the nearest point is and
  in which direction (**+** above you on the band, **−** below).
- **Store → Save** flow, **single-row** inline editing (click a row to edit
  only it — no accidental neighbour edits), and a **confirmed** delete (✓ / ✗).
- **Settings → Tuner** is the full table editor: add / rename antennas,
  delete, clear, and set the "nearest match" window (how close counts as
  *exact*).
- Matches by **dial (carrier) frequency**, so it reads the same in USB or
  LSB — correct, because an antenna tuner matches the RF frequency, not the
  sideband.

## FM transmit polish

- **Pre-emphasis selector** on the Tuning panel — **Off** or **Comm** (the
  6 dB/oct communications curve). Voice defaults to Comm; digital auto-bypasses.
- **FM-aware bandwidth** — the RX/TX bandwidth presets follow the FM signal's
  occupied channel, and the **Deviation** control auto-sizes the RX bandwidth
  to match (so wider deviation opens the receiver to suit).
- **The native mic rack auto-bypasses in FM** — the studio TX DSP chain is
  meant for SSB/ESSB voice, so it steps out of the way on FM.

## Global tooltips on/off (Settings → Visuals)

A single **Show tooltips** switch turns every hover tooltip across the app on
or off — handy once you know your way around.

## Header chip strip tidy-up

The header launcher chips are regrouped: **TX DSP** (mic rack) · **RX DSP**
(RX EQ) · **Options** (CTUN · Tuner · CW · CW Dec · WF-ID). The CW console /
decoder and the Tuner now sit with the other operating tools under
**Options**, and RX DSP moves left next to TX DSP.

## Notes

- **Windows-only** reality is stated in the README and the About chip — Linux /
  macOS are on the roadmap but not yet buildable.
- The Tuner is pure UI + saved settings; it has no effect on the RF / wire
  path (RF-safe).

## Install

`dist/Lyra-Setup-0.7.0.exe` — Windows installer (bundles the Qt runtime and
DSP engine; no separate downloads).
